🎓 EduPulse X
Student Performance Analytics Dashboard
EduPulse X is a full-stack data analytics product that transforms raw student data into actionable insights, risk predictions, and personalized recommendations through an interactive dashboard.
This project combines Data Science + Machine Learning + Product UI (Streamlit) to simulate a real-world EdTech analytics system.

Core Features ->
📊 1. Executive Dashboard
KPI cards (GPA, risk, study time, absences)
cohort-level insights
intervention queue for high-risk students
📈 2. Academic Intelligence
GPA & grade distribution
study time vs performance analysis
attendance impact visualization
correlation heatmaps
model feature importance
⚠️ 3. Risk Intelligence
early warning system
risk score computation
high-risk student identification
intervention pattern analysis
🧠 4. Student Segmentation
clustering-based grouping
behavior-based student personas
cluster-level insights
👤 5. Student Explorer
individual student profile
cohort comparison
personalized recommendations
🔬 6. What-If Lab
simulate changes in:
study time
absences
tutoring
observe impact on:
predicted GPA
risk score
recommendations
